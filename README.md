# PostgreSQL Learning Guide 🐘

A self-study checklist based on the official [PostgreSQL 18 Documentation](https://www.postgresql.org/docs/current/index.html).

## What's inside

- **50 topics** covering the full PostgreSQL docs — from beginner SQL to server internals
- **⭐ 20 key topics** highlighted for practical, real-world database work
- **Direct links** to every chapter in the official docs
- **Suggested learning order** so you know where to start

## How to use it

Open `postgresql_learning_guide.md` and check off topics as you complete them. Start with the ⭐ starred chapters if you want the fastest path to being productive with PostgreSQL.
